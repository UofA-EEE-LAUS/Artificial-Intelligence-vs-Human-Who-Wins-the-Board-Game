****Files overview*****
app.py runs tic-tac-toe using the standard minimax algorithm
a_b_prune.py runs the game using optimised Alpha-beta-prunning.
Home.py runs the game as an app
/documents contain documents related to the project
/heuristics/heuristic_a_b.py plays five in a row using heuristic based alpha beta prunning algorithm and minimax algorithm

****Installations Required*****
Python 3.x, no other installations are required.

****To run the game******
1. Open powershell or command prompt in the current directory.
2. To run game using minimax algorithm, run:
    >python Home.py

****Changing between Tic-tac-toe & 5-in-a-row*****
1. Choose the game from a list shown in the GUI (opened by running the Home.py)
2. Select a depth from a drop down to access five-in-a-row
